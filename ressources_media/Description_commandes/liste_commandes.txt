------------------------------------------------------------------------------
ADD    : Ajouter d'une ressource
LOAD   : Charger un fichier texte contenant les ressources. Le contenu de la mediathèque sera entièrement remplacé
SAVE   : Sauvegarder le contenu des resssources dans un fichier texte
SEARCH : Rechercher une ressource
CLEAR  : Réinitialiser la recherche
LIST   : Afficher les ressources de manière compacte
SHOW   : Afficher les informations d'une ressource
DELETE : Supprimer une resssource
RESET  : Supprimer toutes les ressources
